# Baltazar Chemistry Game

## Overview

Baltazar is a C# educational game developed for the National Research Centers Software Competition in the Laboratory Software category.

The project was designed to present chemistry concepts, especially chemical compounds, through an interactive and game-based learning experience.

## Competition Background

🏆 National Research Centers Software Competition  
**Category:** Laboratory Software

The project was developed as an educational software solution that combines programming, game design, and chemistry concepts.

## Technologies

- C#
- .NET Framework
- Windows Forms

## Features

- Interactive chemistry learning environment
- Game-based approach for learning chemical compounds
- User-friendly Windows desktop interface
- Implementation of educational concepts through software design
